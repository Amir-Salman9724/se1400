Design Philosophy

My approach to design is centered around the user. I believe that a website should not only look modern and clean but should also be intuitive to navigate. Using tools like CSS Grid and Flexbox allows me to create layouts that are flexible and robust. I focus heavily on Information Architecture, ensuring that the most important content is easily accessible. This portfolio serves as a live demo of my ability to structure data and present it in a visually appealing way.


Future Development

As I continue my journey at Utah Tech, I am expanding my portfolio to include full-stack applications. I am currently exploring how to integrate databases with front-end interfaces to create dynamic content. My future goals include working with frameworks like React or Vue to build more complex single-page applications. I am also interested in UI/UX design, specifically how color theory and typography influence user behavior and brand perception.


Commitment to Quality

Every project in this portfolio undergoes rigorous testing. I ensure that all images are optimized for the web to keep loading times low, and I double-check that every link and button functions correctly across different browsers. My commitment to quality means that I am never satisfied with "just enough." I am always looking for ways to refactor my code and improve the user experience, staying up-to-date with the latest industry standards and best practices.
